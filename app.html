API Documentation

Image Generation API

Generate images from text prompts with our simple REST API.

GET https://img.hello-kaiiddo.workers.dev/generate

Parameters

Parameter Type Required Description
prompt string Yes Text description of the image you want to generate
dimensions string No (default: "1:1") Aspect ratio (1:1, 16:9, 9:16, 4:3, 3:4)
image number No (default: 1) Number of images to generate (1-6)
safety boolean No (default: true) Enable/disable content safety filter

Example Request

https://img.hello-kaiiddo.workers.dev/generate?prompt=a%20cat%20wearing%20sunglasses&dimensions=16:9&image=3

Example Response

{
  "images": [
    "https://tmpfiles.org/dl/12345/image1.png",
    "https://tmpfiles.org/dl/12346/image2.png",
    "https://tmpfiles.org/dl/12347/image3.png"
  ],
  "meta": {
    "success": true,
    "parameters": {
      "prompt": "a cat wearing sunglasses",
      "dimensions": "16:9",
      "image": 3
    }
  }
}
