Image Generation API v3

Recommended

Our latest API with enhanced privacy, better quality, and more control. Requests are not visible in network logs.

API Endpoint 
POST https://api.vistaraai.com/v3/generate

Parameters

Parameter Type Required Description
prompt string Yes Text description of the image to generate
model string No Model to use (flux-3d, realistic-vision, anime-diffusion, portrait-plus)
width integer No Width of the image (512-1024)
height integer No Height of the image (512-1024)
seed integer No Random seed for reproducible results

Example Request

JavaScript Fetch 
fetch('https://api.vistaraai.com/v3/generate', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
        'Authorization': 'Bearer YOUR_API_KEY'
    },
    body: JSON.stringify({
        prompt: "A futuristic city at night",
        model: "flux-3d",
        width: 1024,
        height: 1024
    })
})
.then(response => response.json())
.then(data => console.log(data));
