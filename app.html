Image Generation API v1

Original

Our original multi-image generation API with flexible options for aspect ratio and style.

API Endpoint 
GET https://img.hello-kaiiddo.workers.dev/v1/generate

Parameters

Parameter Type Required Description
prompt string Yes Text description of desired image
image number No Model version (1-3)
dimensions string No Aspect ratio (1:1, 16:9, 9:16, 4:3, 3:4)
style string No Art style (photorealistic, anime, painting, digital-art, pixel-art)

Example Request

JavaScript Fetch 
fetch('https://img.hello-kaiiddo.workers.dev/v1/generate?prompt=a+sunset&style=photorealistic')
.then(response => response.json())
.then(data => console.log(data));

Example Response

JSON Response 
{
    "images": [
        "https://example.com/image1.png",
        "https://example.com/image2.png"
    ],
    "meta": {
        "success": true,
        "parameters": {
            "prompt": "a sunset",
            "style": "photorealistic"
        }
    }
}

Image Generation API v2

Simple

Simplified API with automatic temporary hosting of generated images.

API Endpoint 
GET https://img.hello-kaiiddo.workers.dev/v2/generate

Parameters

Parameter Type Required Description
prompt string Yes Text description of desired image

Example Request

JavaScript Fetch 
fetch('https://img.hello-kaiiddo.workers.dev/v2/generate?prompt=a+sunset')
.then(response => response.json())
.then(data => console.log(data));

Example Response

JSON Response 
{
    "status": "success",
    "hosted_url": "https://tmpfiles.org/dl/12345/image.png",
    "prompt": "a sunset",
    "expires_in": "60 minutes"
}

Image Generation API v3

Advanced

Advanced API using backed with human-like request patterns and enhanced privacy.

API Endpoint 
GET https://img.hello-kaiiddo.workers.dev/v3/generate

Parameters

Parameter Type Required Description
prompt string Yes Text description of desired image
width number No Image width (default: 1024)
height number No Image height (default: 1024)
model string No Model to use (flux, realistic-vision, anime-diffusion, portrait-plus)
seed number No Random seed for reproducible results
negative_prompt string No What to exclude from the image
safe boolean No Enable safety filter (default: true)

Example Request

JavaScript Fetch 
fetch('https://img.hello-kaiiddo.workers.dev/v3/generate?prompt=a%20futuristic%20cityscape&width=1024&height=768&model=flux&seed=12345')
.then(response => response.json())
.then(data => console.log(data));

Example Response

JSON Response 
{
    "status": "success",
    "hosted_url": "https://tmpfiles.org/dl/12345/image.png",
    "prompt": "a futuristic cityscape",
    "expires_in": "60 minutes",
    "content_type": "image/webp",
    "size_bytes": 245678,
    "parameters": {
        "width": 1024,
        "height": 768,
        "model": "flux",
        "seed": 12345
    }
}
